🔍 Binance TH Content Checker

ตรวจ กลต. compliance + typo ผ่าน Claude.ai — กดปุ่มเดียวจบ

วิธีใช้ (1 chat = ตรวจได้ไม่จำกัด)

  1. กดปุ่ม Copy Prompt ด้านบน
  2. เปิด claude.ai → New Chat
  3. Paste (Ctrl+V หรือ ⌘+V) → ส่ง
  4. รอ Claude ตอบ "พร้อมตรวจแล้ว"
  5. วาง content หรือแนบไฟล์ (PDF/DOCX/ข้อความ) → ได้ผลตรวจทันที
  6. ตรวจ content อื่นต่อใน chat เดียวกันได้เลย
